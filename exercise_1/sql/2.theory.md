a) What are the main categories of SQL commands?
Man brukar dela in dem i fyra huvudkategorier, men de två första är viktigast för dig just nu:

DDL (Data Definition Language): Kommandon som bygger strukturen (huset).

Exempel: CREATE TABLE, ALTER TABLE, DROP TABLE.

DML (Data Manipulation Language): Kommandon som hanterar datan (möblerna).

Exempel: SELECT, INSERT, UPDATE, DELETE.

(Överkurs) DCL: Rättigheter (vem får logga in?).

(Överkurs) TCL: Transaktioner (spara eller ångra ändringar).

b) Explain the difference between HAVING and WHERE clauses?
Båda används för att filtrera, men de jobbar vid olika tidpunkter:

WHERE: Filtrerar rader innan de grupperas.

Exempel: "Ta bort alla rader där lönen är 0" (innan vi räknar snittet).

HAVING: Filtrerar grupper efter GROUP BY.

Exempel: "Visa bara de länder som har fler än 5 jobb" (vi måste räkna/gruppera först för att veta antalet).

c) How to make sure you delete the correct rows?
Det absolut bästa knepet (som räddar liv!): Kör alltid en SELECT först.

Om du tänker köra:

SQL

DELETE FROM tabell WHERE id = 5;
Kör först:

SQL

SELECT * FROM tabell WHERE id = 5;
Då ser du exakt vad som kommer försvinna. Är det rätt rad? Då kan du byta SELECT * mot DELETE.

d) How do you retrieve unique values in a column?
Du använder nyckelordet DISTINCT.

Exempel: SELECT DISTINCT job_title FROM salaries;

e) What does data transformation mean?
Det betyder att man ändrar datans format, struktur eller värden för att göra den mer användbar eller ren.

Exempel från din övning:

När vi ändrade 'CT' till 'Contract' (med CASE).

När vi räknade om USD till SEK (Matte).

När vi gjorde om lön till texten 'High'/'Low' (Kategorisering).

f) How do you create a new row in a table?
Med kommandot INSERT INTO.

Exempel: INSERT INTO tabell (namn, ålder) VALUES ('Anna', 30);

g) What happens if you omit the WHERE clause in an UPDATE statement?
⚠️ Katastrof! Om du glömmer WHERE uppdateras ALLA rader i hela tabellen.

Exempel: UPDATE salaries SET salary = 20000; -> Nu har alla, även chefen, 20 000 i lön.

h) What happens if you omit the WHERE clause in a DELETE statement?
⚠️ Också katastrof! Om du glömmer WHERE raderas ALLA rader i hela tabellen. Tabellen finns kvar (strukturen), men den är helt tom.

i) What is a conditional statement in SQL, and can it be used with SELECT?
Ett villkorligt uttryck (conditional statement) är CASE (SQL:s version av "If-Else").

Och ja, det används oftast inuti en SELECT för att skapa nya kolumner baserat på logik.

Exempel: "Om lönen är över 50k, skriv 'Hög', annars 'Låg'".
