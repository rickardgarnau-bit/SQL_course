# SQL-Kursstruktur

Detta repository innehåller material för en kurs i SQL, med specifikt fokus på moderna databasfunktioner och DuckDB.

## Kursöversikt (Struktur)

Kursen är uppdelad i följande moduler, baserat på mappstrukturen:

1.  **04_querying_data**
    * Grunderna i att hämta data med `SELECT` från tabeller.

2.  **05_filtering**
    * Filtrera rader med `WHERE`-satsen.

3.  **06_crud_operations**
    * Grundläggande databasoperationer: `CREATE`, `READ`, `UPDATE`, `DELETE`.

4.  **07_generating_data**
    * Metoder för att generera testdata eller dataserier.

5.  **08_grouping_data**
    * Aggregera och gruppera data med `GROUP BY` och funktioner som `COUNT()`, `SUM()`, `AVG()`.

6.  **09_strings**
    * Funktioner och tekniker för att hantera och manipulera textsträngar.

7.  **10_set_operations**
    * Kombinera resultatset med `UNION`, `INTERSECT`, och `EXCEPT`.

8.  **11_joins**
    * Kombinera data från flera tabeller med `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, etc.

9.  **12_cte**
    * Använda `WITH`-satser (Common Table Expressions) för att strukturera komplexa frågor.

10. **13_macros**
    * Skapa och använda makron för att återanvända SQL-kod (specifikt för DuckDB).

11. **14_python_duckdb**
    * Integrera och använda DuckDB direkt från Python-skript.

12. **15_sql_injection**
    * Förstå och förhindra säkerhetsrisker som SQL Injection.

13. **16_duckdb_tricks**
    * Avancerade tips, tricks och optimeringar specifika för DuckDB.
